Demos

Custom Radio Buttons

Selected Value:
Radio A Radio B Radio C 
          Selected Value: <input type="text" ng-model="demoRadio1">
          <div checkgroup="demoRadio1">
            <check value="Radio A">Radio A</check>
            <check value="Radio B">Radio B</check>
            <check value="Radio C">Radio C</check>
          </div>
          <div>
            <button ng-click="select('demoRadio1', ['Radio B'])">Select B</button>
            <button ng-click="select('demoRadio1', ['Radio A','Radio B'])">Select A, B</button>
          </div>

Custom Multiple Check Buttons

Select All Select None A B C

          <div checkgroup="demoMultiple1" check-multiple="true">
            <check check-type="all">Select All</check>
            <check check-type="none">Select None</check>
            <check value="A">A</check>
            <check value="B">B</check>
            <check value="C">C</check>
          </div>
          <div>
            <button ng-click="select('demoMultiple1', ['B'])">Select B</button>
            <button ng-click="select('demoMultiple1', ['A','B'])">Select A, B</button>
          </div>
